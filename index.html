About Me

"Halo, nama saya Faizal Anugrah Pratama. Saya adalah Mahasiswa Universitas Majalengka Jurusan Teknik Informatika. Saya memiliki pengalaman 1 tahun sebagai content creator. Alasan saya masuk teknik informatika karena saya ingin mengetahui teknologi apa saja yang dipakai untuk membuat sebuah website karena cita cita saya menjadi web developer, Ada banyak alasan mengapa menjadi web developer adalah cita-cita saya. Pertama, web developer adalah profesi yang sangat dibutuhkan.

Saat ini, hampir setiap bisnis memiliki website, sehingga permintaan akan web developer selalu tinggi. "Kedua, web developer adalah profesi yang menantang dan bermanfaat. Web developer dapat menggunakan keterampilan mereka untuk membuat sesuatu yang benar-benar baru dan inovatif. Ketiga, web developer adalah profesi yang fleksibel. Web developer dapat bekerja sebagai karyawan, freelancer, atau bahkan mendirikan bisnis sendiri."
